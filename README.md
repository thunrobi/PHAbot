# PHAbot

PHAbot is a web application built for an office robot that acts as a Personal Health Assistant. The system is designed to promote healthier habits in the workplace by encouraging better nutrition, regular exercise, and friendly competition among colleagues.

## Overview

PHAbot interacts with users to:

- Ask what they have eaten
- Ask whether they have worked out
- Suggest healthy meals
- Recommend suitable training methods
- Provide AI-powered chat support
- Display an office health leaderboard

The application is built with Vue.js and provides a responsive, user-friendly interface for interacting with the office robot.

## Features

### Personal Health Tracking
- Daily check-ins about meals and workouts
- Simple input flow for tracking eating habits and exercise

### Smart Recommendations
- Healthy meal suggestions
- Training method recommendations based on user input

### AI Chat
- Integrated AI chat for health-related questions and general support
- Conversational interface for natural interaction

### Office Leaderboard
- Displays rankings within the office
- Encourages friendly competition and healthy lifestyle habits

## Tech Stack

- Vue.js
- JavaScript
- HTML & CSS
- Node.js (for development and build process)
